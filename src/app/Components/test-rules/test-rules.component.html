Test rules

Please read the following instructions very carefully:

  • 1. Each question has 4 options out of which only one is correct.
  • 2. Each question has 60 seconds to answer.
  • 3. You have to finish the test in the specified time
  • 4. Try not to copy or take any sort of help from any sources.
  • 5. You will be awarded 1 mark for each correct answer.
  • 6. There is no penalty for the questions that you have not attempted.
  • 7. There is no penalty for the wrongly answered questions
  • 8. You can change your answer by clicking on some other option.
  • 9. A number list of all questions appears at the middle of the screen, you can access the questions in any order within a section or across sections by clicking on the question number given on the number list.
  • 10. Do not click the button "submit test" before completing the test. A test once submitted cannot be resumed.
  • 11. In case if you leave the test, you can continue from where you stopped.
  • 12. I have read all the instructions carefully and have understood them.
